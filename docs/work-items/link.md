# Link Work Items

## Creating Links

### 1. Open Work Item

1. Navigate to the work item
2. Click **Link** button

### 2. Select Link Type

Choose relationship:

- Relates to
- Blocks
- Is blocked by
- Duplicates
- Is duplicated by

### 3. Select Target Work Item

1. Search for work item
2. Select from results
3. Click **Link**

::: tip
Use appropriate link types to clearly show work item relationships.
:::
