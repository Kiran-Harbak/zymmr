# Search Work Items

## Search Options

### Quick Search

1. Use the search bar
2. Enter keywords
3. Press Enter

### Advanced Search

Use filters for:

- Status
- Type
- Priority
- Assignee
- Date range
- Custom fields

### Saved Searches

1. Configure search parameters
2. Click **Save Search**
3. Name your search
4. Access saved searches anytime

::: tip
Combine multiple filters for more precise results.
:::
