# Log Time

## Time Logging Process

### 1. Access Time Logging

1. Open the work item
2. Click **Log Time** button

### 2. Enter Time Details

Fill in:

- Time spent
- Work date
- Description
- Activity type

### 3. Save Time Log

Click **Save** to record the time entry.

::: warning Note
Time logs cannot be modified after 7 days.
:::
