# Export Work Items

## Export Options

### Available Formats

- CSV
- Excel
- PDF

### Steps to Export

1. Navigate to work items list
2. Select items to export
3. Click **Export** button
4. Choose export format
5. Configure export options
6. Click **Download**

::: tip
Use filters before exporting to include only relevant work items.
:::
